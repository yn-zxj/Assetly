## 简介

Assetly 是一款基于 Tauri 2.x 和 React 19 的跨平台家庭物品管理应用。该项目采用现代前端技术栈，结合 Rust 后端实现，提供桌面端和移动端的本地化资产管理解决方案。

## 系统要求

根据项目配置和 GitHub Actions 工作流，系统要求如下：

### 最低系统要求
- **操作系统**: macOS 10.15+, Windows 10+, Linux (Ubuntu 18.04+)
- **架构**: x64, ARM64 (macOS Universal 支持)
- **内存**: 至少 4GB RAM
- **存储**: 至少 500MB 可用空间

### 开发环境要求
- **Node.js**: LTS 版本 (GitHub Actions 使用 Node.js 20)
- **包管理器**: pnpm (版本 8.x)
- **Rust 工具链**: stable (GitHub Actions 使用 dtolnay/rust-toolchain@stable)
- **Tauri CLI**: 2.x 版本

## 前置条件

### 必需组件
1. **Node.js LTS**: 确保安装最新 LTS 版本
2. **pnpm**: 包管理器，支持工作区和依赖管理
3. **Rust 工具链**: 包含 cargo 和 rustc
4. **Tauri CLI**: Tauri 应用程序的命令行工具

### 平台特定要求

#### macOS
- Xcode Command Line Tools
- macOS 10.15 或更高版本
- Universal 构建支持 (需要 Apple Silicon 和 Intel 双架构)

#### Windows
- Visual Studio Build Tools 或 Visual Studio 2019+
- Windows 10 SDK
- WebView2 Runtime (用于开发环境)

#### Linux
- GTK+ 3 开发库
- WebKitGTK 开发库
- OpenSSL 开发库
- libayatana-appindicator3 (可选，用于系统托盘)

## 安装步骤

### 1. 安装 Node.js 和 pnpm

#### macOS/Linux
```bash
# 使用官方安装脚本
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

# 或使用 nvm (推荐)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
source ~/.bashrc
nvm install --lts
nvm use --lts

# 安装 pnpm
npm install -g pnpm
```

#### Windows
```powershell
# 使用 Chocolatey
choco install nodejs-lts pnpm

# 或使用 Scoop
scoop install nodejs-lts pnpm
```

### 2. 安装 Rust 工具链

#### macOS/Linux
```bash
# 使用 rustup (推荐)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source "$HOME/.cargo/env"

# 或使用 official installer
wget https://static.rust-lang.org/rustup/dist/x86_64-unknown-linux-gnu/rustup-init
chmod +x rustup-init
./rustup-init
```

#### Windows
```powershell
# 下载并安装
curl -O https://static.rust-lang.org/rustup/dist/x86_64-pc-windows-msvc/rustup-init.exe
.\rustup-init.exe
```

### 3. 安装 Tauri CLI

```bash
# 全局安装 Tauri CLI
npm install -g @tauri-apps/cli

# 或使用 pnpm
pnpm add -g @tauri-apps/cli

# 验证安装
tauri --version
```

### 4. 平台特定依赖安装

#### macOS
```bash
# 安装 Xcode Command Line Tools
xcode-select --install

# 安装额外依赖 (可选)
brew install libusb
```

#### Windows
```powershell
# 安装 Visual Studio Build Tools
# 从 https://visualstudio.microsoft.com/downloads/ 下载 "Build Tools for Visual Studio"

# 安装 WebView2 Runtime
# 从 https://developer.microsoft.com/en-us/microsoft-edge/webview2/ 下载
```

#### Linux
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install -y \
    libgtk-3-dev \
    libwebkit2gtk-4.0-dev \
    libayatana-appindicator3-dev \
    libsqlcipher-dev \
    clang \
    cmake \
    pkg-config \
    libssl-dev

# Fedora
sudo dnf install -y \
    gtk3-devel \
    webkit2gtk4.0-devel \
    libappindicator-devel \
    sqlite-devel \
    clang \
    cmake \
    openssl-devel

# Arch Linux
sudo pacman -S --noconfirm \
    gtk3 \
    webkit2gtk \
    libappindicator \
    sqlite \
    clang \
    cmake \
    openssl
```

## 开发服务器启动

### 1. 开发环境启动

```bash
# 启动开发服务器 (推荐)
pnpm tauri dev

# 或分别启动前端和后端
# 在新终端中启动
pnpm dev  # 启动 Vite 开发服务器
```

### 2. 开发服务器配置

开发服务器配置位于 `vite.config.ts` 中：

```typescript
// Vite 配置
export default defineConfig({
  server: {
    port: 1420,
    strictPort: true,
    host: host || false,
    hmr: host ? { protocol: "ws", host, port: 1421 } : undefined,
    watch: {
      ignored: ["**/src-tauri/**"],
    },
  },
});
```

### 3. Tauri 配置

Tauri 开发配置位于 `tauri.conf.json`：

```json
{
  "build": {
    "beforeDevCommand": "pnpm dev",
    "devUrl": "http://localhost:1420",
    "frontendDist": "../dist"
  },
  "app": {
    "windows": [{
      "title": "Assetly",
      "width": 1200,
      "height": 800,
      "minWidth": 375,
      "minHeight": 600
    }]
  }
}
```

## 故障排除

### 1. 常见安装问题

#### Node.js 版本不兼容
```bash
# 检查当前 Node.js 版本
node --version

# 使用 nvm 管理版本
nvm install --lts
nvm use --lts
nvm alias default lts/*
```

#### pnpm 缓存问题
```bash
# 清理 pnpm 缓存
pnpm store prune

# 删除 node_modules 和重新安装
rm -rf node_modules
rm pnpm-lock.yaml
pnpm install
```

#### Rust 工具链问题
```bash
# 更新 Rust 工具链
rustup update

# 重新安装稳定版本
rustup install stable
rustup default stable

# 清理过期组件
rustup component remove --installed-components
rustup component add rustc cargo
```

### 2. 平台特定问题

#### macOS M1/M2 架构问题
```bash
# 检查架构
arch

# 安装 Rosetta (如果需要)
softwareupdate --install-rosetta

# 设置架构偏好
export ARCHFLAGS="-arch x86_64"
```

#### Windows 编译问题
```powershell
# 安装 Visual Studio Build Tools
# 确保选择了 "Desktop development with C++" 工作负载

# 设置环境变量
$env:RUST_BACKTRACE=1
$env:TAURI_PRIVATE_KEY="your-private-key"
```

#### Linux 依赖缺失
```bash
# Ubuntu/Debian
sudo apt-get install -y \
```

## 总结

Assetly 的开发环境搭建相对简单，主要依赖于标准的现代开发工具链。关键要点包括：

1. **环境一致性**: 使用 Node.js LTS、pnpm 和 Rust stable 确保跨平台兼容性
2. **依赖管理**: 采用 pnpm 工作区管理前端和 Rust 依赖
3. **开发体验**: Tauri 的热重载功能提供了优秀的开发体验
4. **平台支持**: 完整支持 macOS、Windows、Linux 和 Android

按照本指南的步骤进行安装和配置，应该能够成功搭建 Assetly 的开发环境。如果遇到任何问题，请参考故障排除部分或查看项目根目录下的 `README.md` 文件获取更多信息。
